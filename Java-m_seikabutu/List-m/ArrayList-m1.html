//ArrayListの宣言
ArrayList<データ型> 変数名 = new ArrayList<>();   //という風に書くよ。
//データ型の部分には、ArrayListの格納する要素のデータ型を指定するよ。
//例えば、文字列の場合はStringを指定するよ。
ArrayListの要素を取得するにはget()、要素数を取得するにはsize()を使うよ。

		//解答例
		A、ArrayList arraylist = new ArrayList<>();
		B、for(int i = 0; i < arraylist.size(); i++) {
			//i番目の要素はmaxより大きければ、maxをi番目の要素に更新する
			if (arraylist.get(i) > max) {
				max = arraylist.get(i);
			}
		}
		//したがってa が正解となります。
		//b は、arraylist[i]と書いているので誤りです。これは配列の要素を表します。正しくは、arraylist.get(i);です
		//c は、arraylist.lengthと書いているので誤りです。これは配列の要素数を表すときに使います。正しくはarraylist.size()です。
