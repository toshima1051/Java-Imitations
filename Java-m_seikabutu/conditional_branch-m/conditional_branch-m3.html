switch文は、
該当するcaseの部分から順に実行され、
breakがくるまで続けるよ。
breakがないと、余計な処理まで実行されることになってしまうよ。
				
//基本構文
switch(整数式){
case 定数式1:
    処理1;
    break;
case 定数式2:
    処理2;
    break;
default:
    処理3:
}
//defaultはどのcaseにも当てはまらなかった場合に実行されるよ。
//c が正しく書かれているため、正解となります。
//a は、(整数式)の部分がないため、誤りとなります。
//b は、breakがないため、余計な処理も実行してしまいます。そのため今回の場合だと
中
大
//という表示になってしまうので、誤りです。
