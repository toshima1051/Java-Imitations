switch文は、
該当するcaseの部分から順に実行され、
breakがくるまで続けるよ。
breakがないと、余計な処理まで実行されることになってしまうよ。
defaultはswitch文の最後に記述するよ。
				
//基本構文
switch(整数式){
case 定数式1:
    処理1;
    break;
case 定数式2:
    処理2;
    break;
default:
    処理3:
}
//defaultはどのcaseにも当てはまらなかった場合に実行されるよ。
//a が正しく書かれているため、正解となります。
//b は、break;がないので、余計な処理も実行します。そのため、誤りです。
//c は、「xyz以外です」と表示するコードがswitch文の外に記述されているため、
//条件に関係なく必ず実行されてしまいます。そのため誤りです。
