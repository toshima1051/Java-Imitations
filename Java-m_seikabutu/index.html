Windowsショートカット

【Alt + Tab】
切り替え。アプリケーションやウィンドウを切り替えることができる。
【Windows + D】
デスクトップ表示/非表示。すぐにデスクトップを表示・非表示することができる。
【Alt + F4】
終了させる。すぐにアプリまたはファイルを終了させる・閉じることができる。
【Win + 右左矢印キー】
ウィンドウ表示分割。アプリやウィンドウを分割して一つの画面にすることができる。
【Win + Tab】
タスクビューを開く。開いているウィンドウの一覧をサムネイル表示することができる。
【Win + E】
エクスプローラーを開く。すぐにエクスプローラーを開くことができる。

Eclipseショートカット

【Ctrl + Space】
コンテンツアシスト。入力を補完することができる。
【Ctrl + Shift + F】
フォーマット。コードのフォーマットを整えることができる。エラーがあると動作しない。
【Ctrl + D】
行の削除。カーソルのある行を削除できる。複数行も可能。
【Ctrl + /】
コメントの切替。コメント化・非コメント化ができる。
【Ctrl + - or Shift + +】
ズームイン・ズームアウト。ソースコードの文字サイズを変更できる。
【Ctrl + Alt + ↑ or ↓】
行を上下にコピー。カーソルのある行を上または下にコピーできる。
